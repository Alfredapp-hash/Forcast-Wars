Use this for marketing pages, App Store descriptions, onboarding hero text, and investor one-pagers.

## 30-second pitch (Arkhe DNA + Attention Cortex)

**Headline:** Your organization’s brain — on your Mac.

**Body:**

Arkhe AgentOS is a native macOS operating system for AI agents — a **living digital organism**. The **Arkhe DNA Interface** is the central experience: an evolving strand where **chromosomes** are agent families and **genes** are individual agents. The **Neural Mesh** overlays live collaboration; the **Activity Monitor** tracks performance. The **Attention Cortex** closes the loop: trends → content → video → publish → measure → dream → evolve the DNA.

You stay in control: approvals for sensitive actions, local-first memory in `~/.arkhe`, and budgets that cap cloud spend. No terminal required after install.

**One-liner:** A living DNA strand for your agent architecture. Synapses for live intelligence. An OS that manufactures attention while you sleep.

## Feature bullets (scannable)

- **Mission Control** — live hierarchy, inter-agent comms feed, telemetry, and kill switch
- **Neural Mesh** — canvas of chromosomes and strands; thick/gold paths = trusted pathways
- **Attention Cortex** — autonomous trend → content → video → publish → analytics → dream loop
- **Forensics & Replay** — scrub any mission, narrate steps, export compliance ZIP + PDF
- **Local-first** — Ollama + Apple Foundation Models; optional cloud escalation with caps
- **Agent Browser** — dual-mode WKWebView with in-browser approval modals

## Pricing tiers (Settings → Economics)

| Tier | What you get |
|------|----------------|
| **Local (Free / Alpha)** | Local models (Ollama, Apple FM), basic memory vault, Mission Control, Neural Mesh view, manual missions. Cloud escalation **off** by default. |
| **Pro (Paid — planned)** | Cloud model escalation within budget caps, Ark Vault sync (Supabase), advanced exports, Attention Cortex publish adapters (YouTube OAuth, analytics ingest), priority support. |
| **Enterprise (Planned)** | Air-gapped deploy, SSO, automated audit export, dedicated support, custom agent registry. |

*Alpha note:* All tiers currently run as private alpha. Economics caps in Settings enforce budgets today; paid billing is not wired yet.
